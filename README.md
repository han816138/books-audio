# 何嘉仁 Go!Go!World 第1冊 CD線上聽 - 仿製網頁

## 專案簡介

這是一個仿製何嘉仁國小英語教材「Go!Go!World 第1冊」CD線上聽網頁的專案。本專案重現了原始網頁的外觀、佈局和互動功能。

## 功能特色

### 已實現的功能

1. **響應式佈局**
   - 清晰的標題區域，包含書籍封面圖片
   - 功能連結按鈕（mp3下載、音軌聆聽表、數位影音頻道）

2. **CD 分頁切換**
   - CD1 和 CD2 分頁切換功能
   - 分頁視覺效果與原始網頁一致

3. **音樂播放器介面**
   - 播放/暫停控制按鈕
   - 進度條顯示
   - 時間顯示（當前時間/總時長）
   - 音量控制圖示
   - 選單圖示

4. **播放模式選擇**
   - 單曲循環模式
   - 全部播放模式

5. **播放速度控制**
   - 慢速播放
   - 正常速度
   - 快速播放

6. **曲目列表**
   - 完整的曲目列表顯示
   - 每首曲目包含：
     - 音樂圖示
     - 曲目名稱
     - 播放時長
     - 曲目編號標籤（右上角）
     - 彩色左邊框（每首曲目不同顏色）
   - 滑鼠懸停效果
   - 點擊曲目可模擬播放

## 技術架構

- **HTML5**: 網頁結構
- **CSS3**: 樣式設計與視覺效果
- **JavaScript (ES6+)**: 互動功能與邏輯
- **Font Awesome**: 圖示庫

```
GGW1/

├── style.css           # 樣式表
```

## 使用說明

1. 開啟 `index.html` 即可在瀏覽器中查看網頁
2. 點擊 CD1 或 CD2 分頁切換不同的曲目列表
3. 點擊曲目可模擬播放功能
4. 使用播放器控制項調整播放模式和速度

## 注意事項

- 本專案為視覺和功能的仿製，不包含實際的音頻檔案
- 播放功能為模擬實現，主要展示 UI 互動效果
- 所有互動功能均已實現，可作為原型或教學示範使用
